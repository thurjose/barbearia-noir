# Barbearia Noir 💈

Landing page fictícia desenvolvida para portfólio, simulando o site de uma barbearia premium.

## 🛠️ Tecnologias

- Next.js
- Tailwind CSS
- TypeScript

## 📄 Páginas e Seções

- **Navbar** — navegação fixa com links âncora
- **Hero** — seção principal com CTA de agendamento
- **Serviços** — cards com preços e descrições
- **Sobre** — história e diferenciais da barbearia
- **Agendamento** — formulário com horários disponíveis
- **Footer** — contato, Instagram e WhatsApp

## 🚀 Como rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`
